 <p>
  封城一个多月，武汉发生了太多在官方媒体上看不到的故事。武汉作家方方坚持写“封城记”，记录下了一座千万人大都市突然沉寂后的点点滴滴。有教授说，几百名记者还抵不上一个方方。
  <br/>
  <br/>
  上周，中共中央政法委下达通知，要求各地在新冠肺炎疫情面前，加大“正能量”宣传，推出更多“有温度、有泪点、有人情味的暖新闻。”但这样的舆论导向遭众多网民炮轰，不少人批评当局是在试图粉饰这场灾难。
 </p>
 <p>
  从1月25日，也就是武汉封城后的第三天，武汉作家方方写下了第一篇“封城记”。之后一个月里，她几乎从未间断。在字里行间中，人们看到了什么才是“人情味”。
  <br/>
  <br/>
  在一篇又一篇的日记当中，方方谈到了她亲友的近况、口罩的紧俏、市面的冷清、灾难的无情。有自媒体人说，方方“太矫情了”，每天散布负面情绪，对抗疫没有任何好处。但方方坚称，她根本没有批评谁的意思。
 </p>
 <p>
  <br/>

 <p>
  她在2月8日的日记中写道：“我绝对与政府保持一致......帮助政府安抚焦虑的人们。只是，我们在方式上各有不同，可能我会在写作过程中，偶尔会冒出自己的感想。”
  <br/>
  <br/>
  在一位国内教授看来，方方的笔墨胜过无数的宣传机器。华中师范大学教授戴建业上周日发文说，只要把中国记者的报道与《方方日记》对比，他们的境界和见识就高下立判了。而中央调往湖北报道疫情的几百名记者加起来，还不如一个方方。他直言，公众看得到的国内媒体报道让人根本读不下去，有些报道甚至是在“侮辱人的智商”。
  <br/>
  <br/>
  2月7日，最早向外界披露新冠肺炎疫情的“吹哨人”之一、武汉眼科医生李文亮病逝，在中国网络世界掀起了一场尤为罕见的“我们要言论自由”的舆论浪潮。听闻他的死讯，方方也加入到了追悼者的行列中。她当天在日记中这样写道：“说出真相的李文亮，受到责罚，丢了性命，到死都没人向他道歉。这样的结果，今后是否还会有人敢说？”
  <br/>
  <br/>
  现居悉尼的中国作家何与怀上周发文说，方方“封城记”中的这些不加修饰的文字是对虚谎的控诉，也是对盛世的诘问，它让人们看到了生命的脆弱、个体的渺小。
