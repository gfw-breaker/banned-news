 <p>
  “共产党没有真实”，这是中国百姓超越了党宣传机器的束缚，与中共政权有过实质性接触、有了切身体会之后，对中共黑权力集团作出的经典总结之一。中国人民的这种经典认识，最近在中共副总理孙春兰到疫区湖北武汉青山区作慰问秀时，被那里已饱受长达六周多封城之苦的人民打开窗户，将“假的！”二字公开大声喊出来。那不断回荡的声音，伴随着让副总理、一众官员尴尬的视频，传到世界各地，在各种社交媒体上引起对中共政权一贯弄虚作假形式主义的热烈讨论。
  <br/>
  <br/>
  这件事情不仅标志着中国人民对于中共政权的本性有了清楚到位的认识，而且能在无任何组织的情况下，在孙春兰到达后，人们自发不约而同开窗，勇敢地向着北京来的高官大喊“假的！”这也表明，人们内心以往固有的对于官员的畏惧之心，正在逐步地被勇气所取代。这正是人民从“觉醒”这个阶段，过渡到第二阶段 - “行动”的重要标志。
  <br/>
  <br/>
  可以想见，当这样的正义之声在全国随处可闻、正义之举在大江南北随处可见时，公民行动就会“从个体行动向有组织地进行战略行动”的第三阶段过渡。那时中国便真正具备了变革的基础，到了“百应具备，只差振臂一呼”的最佳革命时机。
  <br/>
  <br/>
  我想，所有真正与中共政权有过较量的朋友都领教过，这个邪恶政权一旦欺骗不成，就会原形毕露，使出暴力相加的流氓手段。因此，中国的变革恐怕最终无法绕开实力的比拼，这种实力上的较量，不管以什么样的形式、在多大范围内展开，我们都必须事先做好“革命绝不是请客吃饭”的思想准备。
  <br/>
  <br/>
  总之，中国的变革不要寄任何希望在中共政权会遵守规则、协议，或信守承诺上。必须彻底砸烂共产专制的束缚，建立分权制衡的现代政治系统。否则，就不能解决中国的制度性倒行逆施、社会不公、灾祸丛生等现存一切问题。
